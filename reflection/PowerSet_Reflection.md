# Рефлексия PowerSet

Я перепутал и думал, что множество нужно реализовать на основе словаря, а не хэш-таблицы. Из этого вытекают все мои
следующие ошибки:
1. Не параметризовал множество.
2. Добавил лишний статус `PUT_FULL` для метода `put`, так как он уже покрывается статусом `система коллизий не смогла 
найти свободный слот для значения` из хэш-таблицы.
3. Добавил лишний статус `PUT_EXIST` для метода `put`, так как он уже покрывается статусом `система коллизий не смогла
найти свободный слот для значения` из хэш-таблицы.
